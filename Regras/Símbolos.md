## Caracteres não permitidos
Em nomes de variáveis, os caracteres abaixo não são permitidos. Só podem estar presentes em strings literais ou em comentários.
|Character|ANSI Number|Unicode Hex|

|    *    |     42    |  U+002A   |
|    ?    |     63    |  U+003F   |
|    {    |    123    |  U+007B   |
|    }    |    125    |  U+007D   |
|    ª    |    170    |  U+00AA   |
|    º    |    186    |  U+00BA   |
